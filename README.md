# MiniLMS
This project is a Mini Learning Management System (LMS) developed using Django. The system includes an admin panel where the administrator can manage all academic content such as semesters, subjects, chapters, PDF notes, and topic-wise video lectures. The frontend is designed using Bootstrap to provide a simple and responsive user interface. The backend uses PostgreSQL for storing and managing data. On the home page, all semesters are displayed, and each semester contains multiple subjects. Each subject further includes chapters, PDF notes, and videos organized topic-wise, making it easy for users to access learning materials. The project also includes an about page that provides basic information about the system.
